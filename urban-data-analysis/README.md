# Urban Data Analytics Platform

This project is a simple tool I built to explore how urban data can be analyzed to understand patterns and trends.

## Overview
The goal of this project is to allow users to upload datasets and quickly analyze key statistics and visualizations.

## Features
- Upload and preview datasets
- Basic statistical analysis
- Data visualization using histograms
- Simple insights generation

## Tech Stack
- Python
- Pandas
- Streamlit
- Matplotlib

## Why I built this
I wanted to experiment with how data analysis can be applied to real-world systems, especially in areas like urban environments and digital systems.
