# AI Research Assistant

This project is a simple tool I built to explore how research text can be analyzed and structured into useful insights.

## Overview
It takes research text as input and generates summaries, keywords, and structured insights.

## Features
- Summary generation
- Keyword extraction
- Structured analysis (problem, method, results, applications)

## Tech Stack
- Python
- Streamlit

## Why I built this
I wanted to experiment with simplifying research understanding and making technical content easier to process.
